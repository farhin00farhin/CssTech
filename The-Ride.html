This is where the journey started.
MIT brought us all together.
Forming our team
When we were assigned together to create this website, we had no idea how we would work together as a team. Even though we had almost spent an entire semester studying together, we still didn't know a lot about each other. We all had different personalities and backgrounds so communicating with each other wasn't very easy at first.
Working as a team
But once we started working together, it actually turned out to be fun. We got to know each others strength and weaknesses. The journey wasn't always smooth but in the end this project turned out to be one of the best collaborative experiences of our semester.
And finally...
Making this project together wasn't an easy task but with everyone's co operation and contribution, we managed to build a pretty amazing website, given our limited experience with HTML and CSS.
