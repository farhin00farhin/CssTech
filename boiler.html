What's HTML5 Boilerplate?

by Sukhpreet Singh

A template from which one can more easily build a standards-aware website with cross-browser compatibility (Chrome, Firefox, Safari, Opera, IE) even legacy browser support back to IE6.

More Features are

Includes common JavaScript libraries jquery.min.js and modernizr.js Includes common CSS library normalize.min.css Google Analytics tracking code snippet CSS common helper classes, media queries for responsive design Customizable! And open-source!
