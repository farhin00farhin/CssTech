CSS3

-Anita Reddy

Introduction

CSS3 is an extended version of CSS. It is the latest standard used in defining the styles for the different elements of a web page.

CSS3 began to develop in 2001 and some features were initiated only as recently as 2012. CSS3 is backward compatible, which means that your existing CSS2 will remain the same and can be used as it is, without any changes.[1]

In the previous versions of CSS everything was a large single specification describing the various features. The major difference is that CSS3 is split up into modules. Each module has new capabilities to upgrade the web page. Some of the modules are:

  • - Selectors
  • - Animations
  • - Improved backgrounds
  • - Enhanced borders
  • - Multicolumn layout
  • - Additional text effect

How CSS3 Works

CSS works with HTML. While HTML provides structure to a document by organizing information into headers, paragraphs, bulleted lists, and so on, CSS works hand-in-hand with the web browser to make HTML look good. As we know CSS3 is the latest version of CSS2 and it is also compatible with CSS2, so the first thing that needs to be done is to make sure if the older browsers recognize the new elements on our page, so that we can style them. There is no change in how you write CSS. The syntax and rule are same. Only declarations will vary.

What is CSS3 Used For

CSS3 uses more advanced and new features to style your website. For example:

You can also use CSS3 to resize the HTML elements rotate them in all three dimensions and even animate these transformations into smooth transitions.

Additional text effects- new text effects are also available in CSS3 including shadows

CSS3 brings several new ways of defining colours by using primary colours.

Element overflow- overflow is where text is too large to fit within the element and the way overflowing of element contents is handled is vastly improved in CSS3 with numerous options

Improved backgrounds- In CSS3, two new properties were addressed which are background clip and background origin. Between them you can specify where a background should start within an element and how to clip the background. It supports multiple background images and for clipping, changing the origin of a background image.

Enhanced borders- CSS3 brings a lot more flexibility to the way borders can be presented by allowing you to independently change the colours of all four border edges and also to display images for the edges and corners, to place box shadows underneath elements.[2]

Conclusion

CSS3 comprises a huge number of powerful and flexible additions and improvements to CSS, bringing features previously not even possible using Java script.[1]

In my opinion, I totally support the new features of CSS3 and think it’s a very cool when creating a website with whole lot of features and flexibility. CSS3 is here to stay, a tool that is very powerful and can create beautiful, high-quality solutions. Therefore you need to learn them and start using them today, otherwise you will fall behind your competitors.
