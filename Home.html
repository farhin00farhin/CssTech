What's CSS?

CSS = Cascading Style Sheets

CSS lets you work with the fonts, spacing, borders, layout, colour, etc of a website. Basically, CSS is a language to define the format of contents in a website.

CSS lets you maintain the look of a large website with lots of HTML pages. You write your CSS code inside .css files which you can include in any HTML page you want. You can define styles within your HTML, but using CSS makes it easier to read, modify and reuse style information.[1][2]
